# Projeto de cadastro de pessoas em Python

Este é um projeto que implementa um sistema **CRUD** (Create, Read, Update, Delete) para o **cadastro de pessoas** em Python.

### Funcionalidades

O projeto possui as seguintes funcionalidades:

1. **Visualizar dados:** o usuário pode consultar os dados de uma pessoa pelo seu nome, e visualizá-los na tela.

2. **Cadastrar novos usuários:** o usuário pode inserir os dados de uma pessoa (nome e idade) e salvá-los em um arquivo txt.

3. **Editar dados:** o usuário pode editar os dados de um usuário já cadastrada, e atualizá-los no arquivo de registro.

4. **Deletar dados:** o usuário pode remover os dados de um usuário do arquivo de registro.
